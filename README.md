# 🌟 Interactive Bento Grid Portfolio

A modern, premium, and fully responsive Bento Grid portfolio website designed for web designers and frontend developers.

## 🚀 Live Demo
  **https://YogitKumar-a11y.github.io/bento-portfolio/**

---

##  Features

- **Modern Bento Layout:** A beautiful and unique grid structure inspired by Apple and Stripe's design language.
- **100% Responsive Design:** Fully optimized to look stunning on mobile, tablet, and desktop screens.
- **Live Clock Widget:** A JavaScript-powered real-time clock displaying the local time.
- **Smooth Animations:** Premium hover interactions featuring a subtle purple glow effect.

---

##  Tech Stack

- **HTML5:** Semantic structure.
- **Tailwind CSS:** Modern utility-first styling.
- **Vanilla JavaScript:** Dynamic live clock functionality.
- **FontAwesome:** Premium icons integration.

---

## How to Run Locally

1. Clone or download this repository.
2. Open the `index.html` file in any browser (Chrome, Safari, Edge, etc.).
